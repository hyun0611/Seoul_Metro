# Seoul Metro — 서울메트로 최단경로 탐색기

### 서울메트로 1~8호선 역 데이터를 기반으로 출발역과 도착역 사이의 최단경로를 탐색하는 간단한 웹 애플리케이션입니다.
---

## 사용 기술

### C++
- **그래프 자료구조** (인접 리스트) — 역을 노드로, 노선을 엣지로 표현
- **다익스트라 알고리즘** — 거리를 기반으로 최단경로 탐색
- **우선순위 큐** — 효율적인 탐색을 위한 자료구조
- **CSV** — 공공데이터를 기반으로 호선별 역 목록 및 역간 거리 읽기
- **STL** — vector, map, priority_queue, stringstream

### Python
- **Flask** — REST API 서버 구축
- **subprocess** — C++ 실행 파일 호출 및 연동

### Frontend
- **HTML / CSS / JavaScript** — 글래스모피즘 UI 구현
- **Fetch API** — 비동기 경로 요청

---

## 주요 기능

- 출발역 / 도착역 입력 시 최단경로를 탐색
- 환승역 자동 감지 및 색상, 뱃지로 표시
- 호선별 색상으로 경로 시각화
- 총 이동 역 수 / 환승 횟수 표시

---

## 데이터

- 공공데이터포털 서울메트로 지하철 역간 거리 데이터 사용
- 서울시 관할 구간 (1~8호선)

---

## 실행 방법

```bash
# C++ 컴파일
g++ -std=c++17 -o metro main.cpp

# 서버 실행
python3 server.py

# 브라우저에서 접속
http://localhost:3000
```
